# 更新日志

WakeUpScreen 各版本的主要变更记录。

## [3.0.3]

## [3.0.2]

### 主要变更
- 基于 Jetpack Compose 对全部界面进行重构
- 全新视觉设计系统（Material 3 蓝紫花主题）
- 新增通知日志、功能测试、反馈页面
- 扩展多语言支持
- 适配 Android 14 / 15

### 界面与设计
- 重大 UI 艺术重设计 — 暗色森林黎明主题
- 重大 UI 艺术重设计 — 蓝紫花 M3 主题
- 首页与设置页面迁移至 Jetpack Compose
- 所有二级页面迁移至 Jetpack Compose
- 弹窗迁移至 Compose
- SettingRow 支持自定义尾部图标，优化应用图标展示
- 全局 UI 审查与细节优化

### 功能
- 高级设置中新增「仅充电时唤醒」过滤选项
- 新增反馈页面
- 新增功能测试页面
- 新增通知日志页面，记录通知渠道信息（重要级别、声音、震动）
- 新增多语言支持

### 基础设施
- Gradle 升级至 8.9，AGP 升级至 8.7.3，Kotlin 升级至 2.0.21
- 迁移至 JDK 17
- 废弃 API 全面迁移
- 适配 Android 14 / 15
- MMKV 升级至 2.4.0（支持 16KB 页大小）
- 新增 GitHub Actions CI/CD 流程，支持 APK 签名与自动发布
- 开源协议从 MIT 更新为 GPLv3

## [早期版本]

### 历史功能积累
- 白名单 / 应用过滤列表：指定应用才能触发亮屏
- 口袋模式：接近传感器检测到设备在口袋中时不唤醒屏幕
- 睡眠模式：可配置免打扰时间段
- 免打扰（DND）模式感知
- 电池优化引导及快速跳转入口
- 前台服务持久通知
- 调试 / 日志页面，用于排查通知接收问题
- 语言切换支持
- 底部导航结构
- 自定义开关控件
- ViewModel + LiveData 架构
- 接近传感器优化
